# 🥷 Ninja Userbot - Sog'lom taom Edition

**Telegram Userbot** с AI автоответчиком (Mistral AI), поддержкой изображений и автоматическим определением лидов.

## Что это?

Это **юзербот** — работает как ВАШ Telegram аккаунт, а не как бот.

- 🤖 Автоответ на личные сообщения как реальный сотрудник компании
- 📷 **Поддержка изображений через Mistral Vision API**
- 📜 **AI видит историю переписки (контекст) — последние 20 сообщений**
- 🎯 **Автоматическое определение лидов → отправка в Saved Messages**
- 🌐 Веб UI для управления

## Особенности Sog'lom taom Edition

### 🎭 Реалистичный сотрудник
AI отвечает как **Бахром, 35-летний сотрудник** компании Sog'lom taom из Ташкента:
- Родной город — Ташкент, Сергели район
- Знает узбекский и русский языки
- Использует локальные выражения
- Знает всё о компании, ценах, доставке

### 🎯 Автоматические лиды
После каждых 3 сообщений в переписке AI анализирует:
- Готовность клиента заказать
- Запрос расчёта калорий
- Передачу адреса и контактов
- Готовность к оплате

При **успешном лиде** → автоматическое уведомление в **Saved Messages**!

## Использование

1. Откройте `app/web/index.html` в браузере
2. Введите API ID и Hash с [my.telegram.org](https://my.telegram.org)
3. Введите Mistral API Key с [console.mistral.ai](https://console.mistral.ai)
4. Выберите модели:
   - **Vision Model** — для обработки изображений (Pixtral 12B)
   - **Text Model** — для текстовых сообщений (Mistral Medium)
5. Нажмите **Запустить**
6. При первом запуске введите номер телефона и код подтверждения в терминале

## Как это работает

```
[Браузер] <---> [FastAPI :3030] <---> [Telethon] <---> [Telegram]
                                     [Mistral AI Vision]
```

### Обработка сообщений:
1. Telethon получает сообщение (текст и/или изображение)
2. Сообщение + история переписки (до 20 сообщений) отправляются в Mistral AI
   - Если есть изображение → Pixtral (Vision Model)
   - Если только текст → Mistral (Text Model)
3. AI генерирует ответ как сотрудник компании
4. Telethon отправляет ответ от вашего имени

### Определение лидов:
1. Каждые 3 сообщения анализируется переписка
2. AI определяет: является ли это успешным лидом
3. Если да → отправляется уведомление в Saved Messages:

```
🟡 НОВЫЙ ЛИД!

👤 Клиент: Имя клиента
📱 Chat ID: 123456789
📋 Тип: new_client
⏰ Срочность: medium

📝 Что нужно:
Клиент хочет заказать 1400 ккал, нужен расчёт

📊 Уверенность: 85%

🕐 27.04.2026 16:55
```

## API Endpoints

| Endpoint | Method | Описание |
|----------|--------|----------|
| `/api/status` | GET | Статус бота |
| `/api/config` | GET/POST | Конфигурация |
| `/api/start` | POST | Запустить бота |
| `/api/stop` | POST | Остановить бота |
| `/api/logs` | GET/DELETE | Логи сообщений |
| `/api/leads` | GET/DELETE | Логи лидов |
| `/api/history/{chat_id}` | GET | История чата из Telegram |
| `/api/conversation/{chat_id}` | GET | Контекст переписки в памяти |

## Модели Mistral AI

### Vision Models (для изображений)
- **pixtral-12b-2409** — рекомендуется, быстрый и точный

### Text Models (для текста)
- **mistral-medium-latest** — оптимальный баланс

## Данные компании (встроены в промпт)

- Цены: 94,000 - 170,000 сум
- Калории: 1000-3200 ккал
- Доставка: 17:00-22:00, Сергели район
- Карты: Humo, Uzum, Uzcard
- Инстаграм: @soglom.taom

## Безопасность

- Сессия хранится локально в `~/.ninja-data/`
- Изображения временно скачиваются и удаляются после обработки
- Никому не давайте API ID, API Hash и файлы сессии
- Юзербот имеет полный доступ к вашему аккаунту
